
# 🤖 AI Layoff Risk Prediction using Machine Learning & Streamlit

> **A machine learning-powered web application that predicts an employee's layoff risk based on workplace and employment-related factors.**

## 📌 Overview

The rapid adoption of Artificial Intelligence is transforming industries and reshaping job markets. This project aims to analyze employee-related factors and predict the likelihood of layoffs using Machine Learning.

The trained model is deployed through an interactive **Streamlit** web application, allowing users to input employee information and receive an instant layoff risk prediction.

---

## 🎯 Project Objectives

* Predict employee layoff risk using Machine Learning.
* Build an end-to-end ML pipeline from data preprocessing to deployment.
* Provide an easy-to-use web interface for predictions.
* Demonstrate practical implementation of data science concepts.

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/pari00720/AI-layoff-risk-prediction-using-ML-and-Streamlit.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
streamlit run app.py
```

The application will open in your default browser.

---

## 📊 Model Output

The application predicts whether an employee is at a **Low**, **Medium**, or **High** risk of being laid off based on the input features provided by the user.
